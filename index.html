About

A simple site that continiuosly updates you in a set interval when your bus is leaving. It can also say it out loud so you can always stay updated (even while the tab isn't open). It uses the EnTur API that is available anywhere in Norway to get accurate and current updates on bus locations. If available the arrival time will be the estimated using realtime data.
